ParsimonY: structured and sparse machine learning in Python
===========================================================

* `parsimony` provides structured and sparse penalties in machine learning. It currently contains:

    * Penalties:
        * L1 (Lasso)
        * L2 (Ridge)
        * L1+L2 (Elastic net)
        * Total variation (TV)
